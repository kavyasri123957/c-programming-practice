# C Programming Practice

This repository contains a collection of C programming practice programs that cover basic programming concepts, algorithms, and data structures.

## Topics Covered
- Basic C Programs
- Loops (for, while)
- Conditional Statements
- Arrays
- Strings
- Pointers
- Functions
- Structures and Unions
- Dynamic Memory Allocation (malloc, realloc)
- Searching Algorithms
- Sorting Algorithms
- Data Structures (Stack, Queue, Linked List)

## Programs Included
Some of the programs available in this repository:

- Fibonacci series
- Palindrome program
- Reverse a number
- Reverse a string
- Reverse an array
- Swap numbers
- Maximum and minimum in array
- Vowels and consonants checker
- String functions
- Pointer examples
- Linked list implementation
- Stack implementation
- Queue implementation
- Merge sort

## How to Run

Compile the program using GCC:

gcc filename.c

Run the program:

./a.out

or in Windows:

a.exe
